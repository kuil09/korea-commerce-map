# 🇰🇷 대한민국 커머스 지도 (Korea Commerce Map)

대한민국에 존재하는 커머스 서비스들의 특화 카테고리, 상호, 대표 앰블럼, 메인 페이지 링크, 배송 수단, 배송 도착 보장 시간, 최소 주문 금액 등을 한눈에 비교할 수 있는 사이트입니다.

## 🚀 기능

- **카테고리 필터링**: 식료품, 패션, 뷰티, 가전, 퀵커머스 등 다양한 카테고리로 필터링
- **검색 기능**: 서비스 이름, 특징으로 빠른 검색
- **상세 정보 제공**: 배송 방법, 배송 시간, 최소 주문 금액, 주요 특징 등
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두 지원
- **다크 모드**: 시스템 설정에 따른 자동 다크 모드 지원
- **LLM 친화적**: llms.txt 파일로 AI/LLM이 쉽게 정보 접근 가능

## 📁 프로젝트 구조

```
src/
├── app/
│   ├── page.tsx          # 메인 페이지
│   ├── layout.tsx        # 레이아웃
│   └── globals.css       # 전역 스타일
├── components/
│   ├── CommerceList.tsx  # 커머스 리스트 컴포넌트
│   ├── ServiceCard.tsx   # 서비스 카드 컴포넌트
│   ├── CategoryFilter.tsx # 카테고리 필터 컴포넌트
│   └── SearchBar.tsx     # 검색바 컴포넌트
└── data/
    └── commerce.ts       # 커머스 데이터
public/
└── llms.txt              # LLM 친화적 정보 파일
```

## 🛠️ 시작하기

### 요구 사항

- Node.js 18.0 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

[http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

## 🤖 LLM 접근

이 사이트는 LLM(Large Language Model)이 쉽게 정보를 파싱할 수 있도록 `/llms.txt` 파일을 제공합니다. 이 파일에는 모든 커머스 서비스의 핵심 정보가 구조화된 텍스트 형태로 포함되어 있습니다.
